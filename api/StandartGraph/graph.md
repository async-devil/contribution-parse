```
                  <-> Outside width
                    <-----------> Inside Width
                    ↓ StartYPoint
    StartXPoint → --|--|--|--|--| △
                  --|--|--|--|--| |
                  --|--|--|--|--| | Inside height
                  --|--|--|--|--| |
                  --|--|--|--|--| ▽
```

Outside + inside values equal total value
